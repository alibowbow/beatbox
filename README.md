# 🥁 프로페셔널 드럼머신 & 비트박스 생성기

웹 기반 드럼머신과 휴먼 비트박스 생성기입니다. Web Audio API를 사용하여 브라우저에서 직접 드럼 사운드를 생성하고 패턴을 만들 수 있습니다.

## 🎵 주요 기능

### 일반 드럼 모드
- **11가지 드럼 사운드**: 킥, 스네어, 하이햇, 오픈햇, 크래시, 라이드, 탐1, 탐2, 플로어탐, 박수, 카우벨
- **8가지 장르 프리셋**: 록, 재즈, 펑크, 라틴, 힙합, EDM, 레게, 메탈
- **필인 자동 생성**: 스네어 롤, 탐 필, 복합 필인
- **스윙 조절**: 0-70% 스윙감 추가

### 커스텀 비트박스 모드(수정중)
- **사운드 녹음**: 마이크로 직접 비트박스 사운드 녹음
- **커스텀 키 매핑**: A-Z 키에 원하는 사운드 할당
- **샘플 트리밍**: 자동 무음 제거

### 공통 기능
- **16스텝 시퀀서**: 시각적 비트 편집
- **BPM 조절**: 60-200 BPM
- **패턴 저장/불러오기**: JSON 형식
- **드럼 & 베이스 저장소**: 생성 결과를 브라우저에 자동 기록하고 최근 12개를 클릭해 즉시 복원
- **해시 링크 공유**: 드럼 킷·16/24스텝·확률·뮤트·베이스 설정을 `#s=` 링크 하나로 공유
- **실시간 재생**: 스페이스바로 재생/정지
- **베이스 패널 재생 컨트롤**: 드럼과 베이스 루프를 베이스 패널에서도 바로 재생·정지하며 상단 버튼과 상태 동기화
- **베이스 자동 생성**: 모바일에서도 잘 들리는 G2-F#3(높은 Oct 2~낮은 Oct 3) 음역 사용
- **자연스러운 심벌 사운드**: 오픈햇은 단일 보이스로 한 번만 울리고, 크래시는 타격·브론즈 바디·에어·비정수 공진이 차례로 퍼지는 전용 보이스로 합성
- **세로형 홍보 영상 스튜디오**: 작은 `영상` 버튼 안에 접혀 있으며 실제 앱 화면과 Web Audio 마스터 출력을 하나의 영상으로 동시 녹화

## 🎬 홍보 영상 녹화

1. HTTPS 또는 localhost에서 앱을 열고 작은 `영상` 버튼을 펼친 뒤 `세로 영상 녹화`를 누릅니다.
2. 브라우저 공유 창에서 **현재 Beatbox 탭**을 선택합니다. 탭 오디오는 선택하지 않아도 됩니다.
3. 앱이 무음 → 드럼 → 베이스 → 새 비트/베이스 → 저장·해시 공유 → 808/일렉트로 패드 순서로 약 25초간 자동 연주합니다.
4. 완료되면 1080×1920 WebM 영상과 싱크 검증용 JSON 로그가 자동으로 내려받아집니다.

스튜디오는 안내 배너 없이 24px 높이로 접혀 있고, 녹화 시작 전에 문서 흐름에서 자동으로 숨겨져 영상 상단을 가리지 않습니다. 재생음은 `AudioContext.currentTime` 절대 시각으로 예약되고 화면 플레이헤드도 같은 시각을 사용합니다.

## 💾 드럼 & 베이스 저장·공유

- 비트와 베이스가 모두 생성되면 `beatbox-history-v1`에 자동 저장됩니다.
- `내 드럼 & 베이스 저장소`에서 수동 저장, 클릭 복원, 개별 삭제와 전체 비우기를 사용할 수 있습니다.
- 저장소는 기본적으로 얇은 한 줄로 접혀 있어 베이스 그리드와 드럼 패드 사이 흐름을 방해하지 않습니다.
- `링크 복사`는 현재 전체 상태를 UTF-8 Base64URL로 인코딩해 `#s=` 해시에 넣습니다. 링크를 열면 별도 서버 없이 같은 루프가 복원됩니다.
- 같은 상태는 중복으로 쌓이지 않고 최신 항목으로 갱신되며 최대 12개를 유지합니다.
- 보안과 호환성을 위해 공유 데이터를 버전·타입·범위별로 검증합니다. 저장소·해시 공유·JSON 내보내기는 일반 드럼+베이스 모드 전용이며, 마이크로 녹음한 커스텀 `AudioBuffer`는 직렬화하지 않습니다.
- 앱을 `file://`로 직접 연 경우 로컬 파일 경로는 타인에게 유효하지 않으므로 `링크 복사`는 이식 가능한 `#s=` 해시만 복사합니다. 실제 공유에는 배포된 Beatbox 주소 뒤에 이 해시를 붙이세요.

## 🎹 키보드 단축키

### 일반 드럼 모드
- `Q`: 킥 | `W`: 스네어 | `E`: 하이햇 | `R`: 오픈햇 | `T`: 크래시
- `Y`: 라이드 | `A`: 탐1 | `S`: 탐2 | `D`: 플로어탐 | `F`: 박수 | `G`: 카우벨

### 공통
- `Space`: 재생/정지
- `Enter`: 랜덤 비트 생성
- `Backspace`: 패턴 초기화

## 🚀 사용 방법

1. 파일을 브라우저에서 열기
   - `index.html`: 기본 비트박스 생성기
   - `enhanced-beatbox.html`: 프로페셔널 드럼머신

2. 모드 선택
   - 일반 드럼 모드: 즉시 사용 가능한 드럼 사운드
   - 커스텀 모드: 직접 녹음한 사운드 사용

3. 비트 만들기
   - 시퀀서 그리드 클릭으로 비트 추가/제거
   - 장르 프리셋 선택 또는 랜덤 생성
   - 템포와 스윙 조절

## ✅ 브라우저 스모크 테스트

```bash
npm install
CHROMIUM_PATH=/path/to/chromium npm test
CHROMIUM_PATH=/path/to/chromium npm run test:promo
```

기본 테스트는 오디오 예약 간격, 정지 시 미래 히트 취소, 자동 베이스 음역, 저장 중복 제거와 24스텝/808/해시 복원을 확인합니다. 전체 promo 테스트는 25초 공연의 무정지 전환, 저장·해시 공유 시연, 상단 패널 숨김, 첫 드럼 0.20초 및 MIDI 43–54 조건까지 검사합니다.
